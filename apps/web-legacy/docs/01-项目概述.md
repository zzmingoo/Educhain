## 1. 项目介绍

EduChain 前端 Legacy 版本是基于 React 18 + Vite 构建的单页应用（SPA），为 EduChain 区块链教育知识共享平台提供用户界面。

### 1.1 项目定位

- **类型**: 单页应用 (SPA)
- **用户群体**: 学习者、教育工作者、管理员
- **核心功能**: 知识分享、区块链存证、智能检索
- **开发周期**: 2025年3月 - 2025年12月

### 1.2 为什么停止维护？

在 2025年底，我们决定使用 Next.js + Intlayer 重构前端，原因如下：

1. **SEO 需求**: SPA 对搜索引擎不友好，需要 SSR/SSG
2. **国际化**: Intlayer 提供更好的多语言支持
3. **性能优化**: Next.js 提供更好的性能和用户体验
4. **现代化**: 采用最新的 React 19 和 Next.js 15

## 12. 总结

EduChain 前端 Legacy 版本是一个功能完整、技术成熟的 React 应用。虽然已停止维护，但其代码质量和架构设计仍具有参考价值。

新版本在此基础上进行了全面升级，采用 Next.js + Intlayer 架构，提供更好的性能、SEO 和国际化支持。
