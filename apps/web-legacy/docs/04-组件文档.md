## 9. 组件使用示例

### 9.1 KnowledgeCard 使用

```typescript
import { KnowledgeCard } from '@/components/knowledge';

function MyPage() {
  const knowledge = {
    id: 1,
    title: '知识标题',
    content: '知识内容...',
    // ...
  };

  return (
    <KnowledgeCard
      knowledge={knowledge}
      onClick={() => navigate(`/knowledge/${knowledge.id}`)}
      showActions={true}
    />
  );
}
```

### 9.2 SearchBar 使用

```typescript
import { SearchBar } from '@/components/search';

function MyPage() {
  const handleSearch = (keyword: string) => {
    console.log('搜索:', keyword);
  };

  return (
    <SearchBar
      placeholder="搜索知识内容..."
      onSearch={handleSearch}
      showSuggestions={true}
    />
  );
}
```

### 9.3 Loading 使用

```typescript
import { Loading } from '@/components/common';

function MyPage() {
  const [loading, setLoading] = useState(true);

  if (loading) {
    return <Loading tip="加载中..." />;
  }

  return <div>内容</div>;
}
```

## 10. 组件开发规范

### 10.1 组件结构

```typescript
// 1. 导入
import { FC } from 'react';
import styles from './MyComponent.module.css';

// 2. 类型定义
interface MyComponentProps {
  // props 定义
}

// 3. 组件实现
export const MyComponent: FC<MyComponentProps> = (props) => {
  // 组件逻辑
  return (
    // JSX
  );
};

// 4. 默认导出（可选）
export default MyComponent;
```

### 10.2 Props 设计原则

- 必需的 props 不使用可选标记
- 提供合理的默认值
- 使用 TypeScript 类型约束
- 回调函数使用 `on` 前缀

### 10.3 样式规范

- 使用 CSS Modules 或 styled-components
- 避免内联样式
- 使用 CSS 变量
- 遵循 BEM 命名规范
