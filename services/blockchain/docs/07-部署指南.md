# 部署指南

## 1. 环境要求

### 1.1 系统要求

| 组件 | 最低要求 | 推荐配置 |
|------|---------|---------|
| 操作系统 | Linux/macOS/Windows | Ubuntu 20.04+ |
| Python | 3.11+ | 3.11+ |
| 内存 | 512MB | 2GB+ |
| 磁盘 | 1GB | 10GB+ |
| CPU | 1核 | 2核+ |

### 1.2 依赖软件

- Python 3.11+
- pip（Python 包管理器）
- virtualenv（可选，推荐）
- Docker（容器化部署）
- PostgreSQL/MySQL（生产环境）

## 2. 本地开发部署

### 2.1 克隆代码

```bash
# 克隆项目
git clone https://github.com/zzmingoo/educhain.git
cd educhain/services/blockchain
```

### 2.2 创建虚拟环境

```bash
# 创建虚拟环境
python3 -m venv venv

# 激活虚拟环境
# Linux/macOS
source venv/bin/activate

# Windows
venv\Scripts\activate
```

### 2.3 安装依赖

```bash
# 安装依赖包
pip install -r requirements.txt

# 验证安装
pip list
```

### 2.4 配置环境变量

创建 `.env` 文件：

```bash
# 数据库配置
DATABASE_URL=sqlite:///blockchain.db

# 服务配置
HOST=0.0.0.0
PORT=8000

# 基础URL
BASE_URL=http://localhost:8000

# CORS配置
CORS_ORIGINS=http://localhost:3000,http://localhost:8080

# 日志级别
LOG_LEVEL=INFO
```

### 2.5 启动服务

```bash
# 方式一：使用 main.py
python main.py

# 方式二：直接使用 uvicorn
uvicorn app.api:app --host 0.0.0.0 --port 8000 --reload

# 方式三：生产模式（多进程）
uvicorn app.api:app --host 0.0.0.0 --port 8000 --workers 4
```

### 2.6 验证服务

```bash
# 健康检查
curl http://localhost:8000/health

# 查看 API 文档
open http://localhost:8000/docs
```

## 3. Docker 部署

### 3.1 构建镜像

```bash
# 构建 Docker 镜像
docker build -t educhain-blockchain:latest .

# 查看镜像
docker images | grep educhain-blockchain
```

### 3.2 运行容器

```bash
# 运行容器
docker run -d \
  --name blockchain-service \
  -p 8000:8000 \
  -e DATABASE_URL=sqlite:///blockchain.db \
  -e BASE_URL=http://localhost:8000 \
  -v $(pwd)/data:/app/data \
  educhain-blockchain:latest

# 查看日志
docker logs -f blockchain-service

# 停止容器
docker stop blockchain-service

# 删除容器
docker rm blockchain-service
```

### 3.3 Docker Compose

创建 `docker-compose.yml`：

```yaml
version: '3.8'

services:
  blockchain-service:
    build: .
    container_name: blockchain-service
    ports:
      - "8000:8000"
    environment:
      - DATABASE_URL=postgresql://user:password@db:5432/educhain_blockchain
      - BASE_URL=http://localhost:8000
      - CORS_ORIGINS=*
    volumes:
      - ./data:/app/data
      - ./certificates:/app/certificates
      - ./qrcodes:/app/qrcodes
    depends_on:
      - db
    restart: unless-stopped

  db:
    image: postgres:15
    container_name: blockchain-db
    environment:
      - POSTGRES_USER=user
      - POSTGRES_PASSWORD=password
      - POSTGRES_DB=educhain_blockchain
    volumes:
      - postgres_data:/var/lib/postgresql/data
    restart: unless-stopped

volumes:
  postgres_data:
```

启动服务：

```bash
# 启动所有服务
docker-compose up -d

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down

# 停止并删除数据
docker-compose down -v
```

## 4. 生产环境部署

### 4.1 使用 PostgreSQL

```bash
# 安装 PostgreSQL
sudo apt-get install postgresql postgresql-contrib

# 创建数据库和用户
sudo -u postgres psql
CREATE DATABASE educhain_blockchain;
CREATE USER blockchain_user WITH PASSWORD 'your_password';
GRANT ALL PRIVILEGES ON DATABASE educhain_blockchain TO blockchain_user;
\q

# 配置环境变量
export DATABASE_URL=postgresql://blockchain_user:your_password@localhost:5432/educhain_blockchain
```

### 4.2 使用 Nginx 反向代理

创建 Nginx 配置 `/etc/nginx/sites-available/blockchain`:

```nginx
server {
    listen 80;
    server_name blockchain.educhain.com;

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    # 静态文件
    location /certificates/ {
        alias /path/to/educhain/services/blockchain/certificates/;
    }

    location /qrcodes/ {
        alias /path/to/educhain/services/blockchain/qrcodes/;
    }
}
```

启用配置：

```bash
# 创建软链接
sudo ln -s /etc/nginx/sites-available/blockchain /etc/nginx/sites-enabled/

# 测试配置
sudo nginx -t

# 重启 Nginx
sudo systemctl restart nginx
```

### 4.3 使用 Systemd 管理服务

创建服务文件 `/etc/systemd/system/blockchain-service.service`:

```ini
[Unit]
Description=EduChain Blockchain Service
After=network.target

[Service]
Type=simple
User=www-data
WorkingDirectory=/path/to/educhain/services/blockchain
Environment="PATH=/path/to/educhain/services/blockchain/venv/bin"
Environment="DATABASE_URL=postgresql://user:password@localhost:5432/educhain_blockchain"
ExecStart=/path/to/educhain/services/blockchain/venv/bin/uvicorn app.api:app --host 0.0.0.0 --port 8000 --workers 4
Restart=always

[Install]
WantedBy=multi-user.target
```

管理服务：

```bash
# 重新加载 systemd
sudo systemctl daemon-reload

# 启动服务
sudo systemctl start blockchain-service

# 设置开机自启
sudo systemctl enable blockchain-service

# 查看状态
sudo systemctl status blockchain-service

# 查看日志
sudo journalctl -u blockchain-service -f
```

### 4.4 使用 Supervisor

安装 Supervisor：

```bash
sudo apt-get install supervisor
```

创建配置文件 `/etc/supervisor/conf.d/blockchain-service.conf`:

```ini
[program:blockchain-service]
command=/path/to/venv/bin/uvicorn app.api:app --host 0.0.0.0 --port 8000 --workers 4
directory=/path/to/blockchain-service
user=www-data
autostart=true
autorestart=true
redirect_stderr=true
stdout_logfile=/var/log/blockchain-service.log
```

管理服务：

```bash
# 更新配置
sudo supervisorctl reread
sudo supervisorctl update

# 启动服务
sudo supervisorctl start blockchain-service

# 查看状态
sudo supervisorctl status

# 重启服务
sudo supervisorctl restart blockchain-service
```

## 6. 数据备份

### 6.1 数据库备份

```bash
# PostgreSQL 备份脚本
#!/bin/bash
BACKUP_DIR="/backup/blockchain"
DATE=$(date +%Y%m%d_%H%M%S)
FILENAME="blockchain_${DATE}.sql"

pg_dump -U blockchain_user educhain_blockchain > ${BACKUP_DIR}/${FILENAME}
gzip ${BACKUP_DIR}/${FILENAME}

# 保留最近7天的备份
find ${BACKUP_DIR} -name "blockchain_*.sql.gz" -mtime +7 -delete
```

### 6.2 文件备份

```bash
# 备份证书和二维码
#!/bin/bash
BACKUP_DIR="/backup/files"
DATE=$(date +%Y%m%d)

tar -czf ${BACKUP_DIR}/certificates_${DATE}.tar.gz certificates/
tar -czf ${BACKUP_DIR}/qrcodes_${DATE}.tar.gz qrcodes/

# 保留最近30天的备份
find ${BACKUP_DIR} -name "*.tar.gz" -mtime +30 -delete
```

## 7. 故障排查

### 7.1 常见问题

**问题1：服务无法启动**
```bash
# 检查端口占用
lsof -i :8000

# 检查日志
tail -f /var/log/blockchain-service.log

# 检查依赖
pip list | grep fastapi
```

**问题2：数据库连接失败**
```bash
# 测试数据库连接
psql -U blockchain_user -d educhain_blockchain -h localhost

# 检查环境变量
echo $DATABASE_URL

# 查看数据库日志
sudo tail -f /var/log/postgresql/postgresql-15-main.log
```

**问题3：PDF 生成失败**
```bash
# 检查目录权限
ls -la certificates/
chmod 755 certificates/

# 检查 ReportLab 安装
python -c "import reportlab; print(reportlab.Version)"
```

### 8.2 应用监控

```python
# 添加性能监控
import time
from fastapi import Request

@app.middleware("http")
async def add_process_time_header(request: Request, call_next):
    start_time = time.time()
    response = await call_next(request)
    process_time = time.time() - start_time
    response.headers["X-Process-Time"] = str(process_time)
    return response
```
