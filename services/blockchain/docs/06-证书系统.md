# 证书系统设计文档

## 1. 证书系统概述

### 1.1 系统目标

为已存证的教育内容生成专业的、可验证的、防伪的 PDF 证书，提供：
- 视觉专业性：符合证书规范的设计
- 可验证性：二维码扫描验证
- 防伪性：区块链技术保证真实性
- 易分享性：PDF 格式便于传播

### 1.2 核心组件

```
证书系统
├── 证书数据生成（certificate.py）
│   ├── 证书编号生成
│   ├── 验证URL生成
│   └── 数据验证
├── PDF 生成（certificate_pdf.py）
│   ├── 页面布局
│   ├── 内容绘制
│   └── 水印添加
└── 二维码生成（qr_code.py）
    ├── URL 编码
    ├── 图片生成
    └── 文件管理
```

## 2. 证书数据结构

### 2.1 CertificateData 类

```python
@dataclass
class CertificateData:
    certificate_id: str          # 证书编号
    knowledge_id: int            # 知识ID
    knowledge_title: str         # 知识标题
    user_id: int                 # 用户ID
    user_name: str               # 用户名
    content_hash: str            # 内容哈希
    block_index: int             # 区块索引
    block_hash: str              # 区块哈希
    timestamp: str               # 存证时间
    verification_url: str        # 验证地址
    qr_code_data: Optional[str]  # 二维码数据
```

### 2.2 证书编号规则

**格式**: `CERT-YYYY-NNNNNN`

- `CERT`: 固定前缀，标识证书类型
- `YYYY`: 4位年份（如 2024）
- `NNNNNN`: 6位唯一编号（000000-999999）

**生成算法**:
```python
def generate_certificate_id(self, knowledge_id: int, block_index: int) -> str:
    year = datetime.now().year
    unique_str = f"{knowledge_id}-{block_index}-{uuid.uuid4().hex[:8]}"
    hash_obj = hashlib.sha256(unique_str.encode())
    unique_number = int(hash_obj.hexdigest()[:6], 16) % 1000000
    
    return f"CERT-{year}-{unique_number:06d}"
```

**示例**:
- `CERT-2024-123456`
- `CERT-2024-789012`
- `CERT-2025-000001`

### 2.3 验证URL生成

**格式**: `{base_url}/blockchain/certificates/{certificate_id}/verify`

**示例**:
```
http://localhost:8000/blockchain/certificates/CERT-2024-123456/verify
https://educhain.com/blockchain/certificates/CERT-2024-123456/verify
```

### 3.2 绘制流程

```python
def generate_pdf(self, certificate: CertificateData, qr_image_path: str):
    c = canvas.Canvas(filepath, pagesize=A4)
    
    # 1. 添加水印
    self._draw_watermark(c)
    
    # 2. 绘制边框
    c.setStrokeColor(colors.HexColor("#1890ff"))
    c.setLineWidth(2)
    c.rect(20*mm, 20*mm, width-40*mm, height-40*mm)
    
    # 3. 绘制头部
    y = self._draw_header(c, y_start)
    
    # 4. 绘制证书信息
    y = self._draw_certificate_info(c, certificate, y)
    
    # 5. 绘制二维码
    y = self._draw_qr_code(c, qr_image_path, y)
    
    # 6. 绘制底部
    self._draw_footer(c, y)
    
    c.save()
```

## 4. 二维码生成

### 4.1 QR Code 配置

```python
qr = qrcode.QRCode(
    version=1,                                    # 大小版本（1-40）
```

**容错级别**:

|------|--------|------|

| M | 15% | 中容错 |
| Q | 25% | 较高容错 |
| H | 30% | 高容错（推荐） |

```python
def generate_certificate_qr(self, verification_url: str, certificate_id: str):
    # 1. 创建 QR Code 对象
```

### 4.3 文件管理

**存储位置**: `qrcodes/` 目录

**命名规则**: `cert_{certificate_id}.png`

**清理策略**:
```python
def cleanup_old_qrcodes(self, max_age_days: int = 30):
    """清理超过30天的旧二维码"""
    current_time = time.time()
    max_age_seconds = max_age_days * 24 * 60 * 60
    
    for filename in os.listdir(self.output_dir):
        filepath = os.path.join(self.output_dir, filename)
        if os.path.isfile(filepath):
            file_age = current_time - os.path.getmtime(filepath)
            if file_age > max_age_seconds:
                os.remove(filepath)
```

## 5. 证书生成流程

### 5.1 完整流程

```
1. 接收证书请求
   ├─ knowledge_id
   ├─ knowledge_title
   ├─ user_id
   └─ user_name

2. 查找区块链记录
   ├─ 根据 knowledge_id 查找交易
   ├─ 查找交易所在区块
   └─ 获取 content_hash, block_hash

3. 生成证书数据
   ├─ 生成证书编号
   ├─ 生成验证URL
   └─ 创建 CertificateData 对象

4. 生成二维码
   ├─ 编码验证URL
   ├─ 生成 PNG 图片
   └─ 保存到 qrcodes/ 目录

5. 生成 PDF 证书
   ├─ 创建 Canvas
   ├─ 绘制各个部分
   └─ 保存到 certificates/ 目录

6. 返回证书信息
   ├─ certificate_id
   ├─ pdf_url
   ├─ qr_code_url
   └─ verification_url
```

```python
def verify_certificate(certificate_id: str):
    # 1. 检查证书文件是否存在
    pdf_path = f"certificates/{certificate_id}.pdf"
    if not os.path.exists(pdf_path):
        return {"valid": False, "message": "Certificate not found"}
    
    # 2. 验证证书编号格式
    if not certificate_id.startswith("CERT-"):
        return {"valid": False, "message": "Invalid format"}
    
    # 3. 查询区块链记录（可选）
    # 验证区块链上是否有对应的存证记录
    
    return {
        "valid": True,
        "certificate_id": certificate_id,
        "message": "Certificate is valid"
    }
```

### 9.3 数字签名

```python
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives.asymmetric import rsa, padding

def sign_certificate(certificate: CertificateData, private_key):
    """为证书添加数字签名"""
    message = certificate.to_json().encode()
    signature = private_key.sign(
        message,
        padding.PSS(
            mgf=padding.MGF1(hashes.SHA256()),
            salt_length=padding.PSS.MAX_LENGTH
        ),
        hashes.SHA256()
    )
    return signature
```
