# 安全设计文档

## 1. 安全概述

### 1.1 安全目标

- **数据完整性**：确保区块链数据不可篡改
- **内容真实性**：验证内容的原创性和时间戳
- **接口安全**：防止恶意请求和攻击
- **隐私保护**：保护用户敏感信息

### 1.2 威胁模型

| 威胁类型 | 风险等级 | 防护措施 |
|---------|---------|---------|
| 数据篡改 | 高 | 哈希链验证 |
| SQL注入 | 中 | ORM参数化查询 |
| DDoS攻击 | 中 | 限流、负载均衡 |
| 未授权访问 | 中 | API认证（待实现） |
| 数据泄露 | 低 | 只存储哈希值 |

## 2. 数据安全

### 2.1 哈希算法

使用 **SHA-256** 算法确保数据完整性：

```python
import hashlib

def calculate_hash(data: str) -> str:
    """计算SHA-256哈希值"""
    return hashlib.sha256(data.encode('utf-8')).hexdigest()
```

**SHA-256 特性**：
- 输出长度：256位（64位十六进制）
- 抗碰撞性：几乎不可能找到两个不同输入产生相同输出
- 单向性：无法从哈希值反推原始数据
- 雪崩效应：输入微小变化导致输出完全不同

### 2.2 区块链完整性

**哈希链保护**：

```python
def is_chain_valid(self) -> bool:
    """验证区块链完整性"""
    for i in range(1, len(self.chain)):
        current_block = self.chain[i]
        previous_block = self.chain[i - 1]
        
        # 验证当前区块哈希
        if current_block.hash != current_block.calculate_hash():
            return False
        
        # 验证哈希链连接
        if current_block.previous_hash != previous_block.hash:
            return False
    
    return True
```

**防篡改机制**：
1. 每个区块包含前一区块的哈希值
2. 修改任何历史区块会导致后续所有区块验证失败
3. 定期验证整条链的完整性

### 2.3 内容隐私

**只存储哈希值**：

```python
# 不存储完整内容
content = "这是一篇教育文章的完整内容..."

# 只存储哈希值
content_hash = hashlib.sha256(content.encode()).hexdigest()

transaction = Transaction(
    type="KNOWLEDGE_CERTIFICATION",
    content_hash=content_hash,  # 只存储哈希
    # 不存储 content 本身
)
```

**优势**：
- 保护内容隐私
- 减少存储空间
- 验证时重新计算哈希比对

## 3. 接口安全

### 3.1 输入验证

使用 **Pydantic** 进行严格的输入验证：

```python
from pydantic import BaseModel, Field, validator

class CertifyRequest(BaseModel):
    type: str = Field(..., regex="^(KNOWLEDGE_CERTIFICATION|ACHIEVEMENT|COPYRIGHT)$")
    knowledge_id: Optional[int] = Field(None, ge=1)
    user_id: int = Field(..., ge=1)
    content_hash: str = Field(..., regex="^[a-f0-9]{64}$")
    
    @validator('content_hash')
    def validate_hash(cls, v):
        if len(v) != 64:
            raise ValueError('Hash must be 64 characters')
        return v.lower()
```

**验证规则**：
- 类型检查：确保字段类型正确
- 格式验证：使用正则表达式验证格式
- 范围检查：数值范围限制
- 自定义验证：复杂业务逻辑验证

### 3.2 SQL 注入防护

使用 **SQLAlchemy ORM** 自动防护：

```python
# 安全：使用 ORM
session.query(BlockModel).filter_by(index=user_input).first()

# 安全：参数化查询
session.query(BlockModel).filter(BlockModel.index == user_input).first()

# 危险：字符串拼接（避免）
# query = f"SELECT * FROM blocks WHERE index = {user_input}"
```

## 4. 认证与授权（未来增强）

### 4.1 JWT 认证

```python
from fastapi import Depends, HTTPException, status
from fastapi.security import HTTPBearer, HTTPAuthorizationCredentials
import jwt

security = HTTPBearer()

def verify_token(credentials: HTTPAuthorizationCredentials = Depends(security)):
    """验证JWT令牌"""
    try:
        payload = jwt.decode(
            credentials.credentials,
            SECRET_KEY,
            algorithms=["HS256"]
        )
        return payload
    except jwt.ExpiredSignatureError:
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED,
            detail="Token expired"
        )
    except jwt.InvalidTokenError:
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED,
            detail="Invalid token"
        )

@app.post("/api/blockchain/certify")
async def certify(
    request: CertifyRequest,
    user=Depends(verify_token)
):
    # 只有认证用户才能访问
    pass
```

### 4.3 权限控制

```python
from enum import Enum

class Permission(str, Enum):
    READ = "read"
    WRITE = "write"
    ADMIN = "admin"

def require_permission(permission: Permission):
    """权限检查装饰器"""
    def decorator(func):
        async def wrapper(*args, user=None, **kwargs):
            if not user or permission not in user.get("permissions", []):
                raise HTTPException(
                    status_code=403,
                    detail="Insufficient permissions"
                )
            return await func(*args, user=user, **kwargs)
        return wrapper
    return decorator

@app.post("/api/blockchain/certify")
@require_permission(Permission.WRITE)
async def certify(request: CertifyRequest, user=Depends(verify_token)):
    # 需要写权限
    pass
```

## 5. 数字签名（未来增强）

### 5.1 RSA 签名

```python
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives.asymmetric import rsa, padding
from cryptography.hazmat.primitives import serialization

# 生成密钥对
private_key = rsa.generate_private_key(
    public_exponent=65537,
    key_size=2048
)
public_key = private_key.public_key()

def sign_transaction(transaction: Transaction, private_key) -> bytes:
    """对交易进行数字签名"""
    message = transaction.to_json().encode()
    signature = private_key.sign(
        message,
        padding.PSS(
            mgf=padding.MGF1(hashes.SHA256()),
            salt_length=padding.PSS.MAX_LENGTH
        ),
        hashes.SHA256()
    )
    return signature

def verify_signature(transaction: Transaction, signature: bytes, public_key) -> bool:
    """验证数字签名"""
    message = transaction.to_json().encode()
    try:
        public_key.verify(
            signature,
            message,
            padding.PSS(
                mgf=padding.MGF1(hashes.SHA256()),
                salt_length=padding.PSS.MAX_LENGTH
            ),
            hashes.SHA256()
        )
        return True
    except Exception:
        return False
```

### 5.2 ECDSA 签名

```python
from cryptography.hazmat.primitives.asymmetric import ec

# 生成 ECDSA 密钥对
private_key = ec.generate_private_key(ec.SECP256R1())
public_key = private_key.public_key()

def sign_with_ecdsa(data: bytes, private_key) -> bytes:
    """使用ECDSA签名"""
    signature = private_key.sign(
        data,
        ec.ECDSA(hashes.SHA256())
    )
    return signature

def verify_ecdsa(data: bytes, signature: bytes, public_key) -> bool:
    """验证ECDSA签名"""
    try:
        public_key.verify(
            signature,
            data,
            ec.ECDSA(hashes.SHA256())
        )
        return True
    except Exception:
        return False
```

## 6. 数据加密

### 6.1 敏感数据加密

```python
from cryptography.fernet import Fernet

# 生成密钥
key = Fernet.generate_key()
cipher = Fernet(key)

def encrypt_data(data: str) -> bytes:
    """加密数据"""
    return cipher.encrypt(data.encode())

def decrypt_data(encrypted_data: bytes) -> str:
    """解密数据"""
    return cipher.decrypt(encrypted_data).decode()

# 使用示例
encrypted = encrypt_data("sensitive information")
decrypted = decrypt_data(encrypted)
```

### 6.2 数据库加密

```python
from sqlalchemy import Column, LargeBinary

class EncryptedBlockModel(Base):
    __tablename__ = 'encrypted_blocks'
    
    id = Column(Integer, primary_key=True)
    encrypted_data = Column(LargeBinary)  # 加密存储
    
    def set_data(self, data: dict, cipher: Fernet):
        """加密并存储数据"""
        json_data = json.dumps(data)
        self.encrypted_data = cipher.encrypt(json_data.encode())
    
    def get_data(self, cipher: Fernet) -> dict:
        """解密并返回数据"""
        decrypted = cipher.decrypt(self.encrypted_data)
        return json.loads(decrypted.decode())
```

from fastapi import Request
from fastapi.responses import JSONResponse

@app.exception_handler(Exception)
async def global_exception_handler(request: Request, exc: Exception):
